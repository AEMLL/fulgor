# Fulgor

Software synth for a Raspberry Pi platform, developed using the following resources:

* *The Audio Programming Book* by Richard Boulanger and Victor Lazzarini
* *Audio Effects: Theory, Implementation and Application* by Andrew McPherson and Joshua Reiss
* *Real-time Audio Programming with Bela* Course by Andrew McPherson  

# Current Features

* Wavetable oscillator source
* Wavetable LFOs
* Low-pass resonant filter
* Delay-line audio effects such as:
    * Delay
    * Chorus
    * Flanger
* MIDI keyboard support, including parameter modulation with CC messages.
